# Ami_Coding_Pari_Na

Clone the repo:
```bash
git clone
cd Ami_Coding_pari_Na
```

create virtual environment
```bash
python -m venv .env
```
Activate it
```bash
source .env/bin/activate
```

```bash
pip install -r requirements.txt
```
Run project
```bash
python manage.py runserver
```
